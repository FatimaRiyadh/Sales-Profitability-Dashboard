# Sales-Profitability-Dashboard

A dynamic Power BI dashboard designed to analyze corporate sales performance and branch profitability to support data-driven managerial decisions.

## Key Project Phases:
1. **Data Cleaning & ETL:** Prepared and cleaned raw records using Power Query.
2. **Data Modeling & DAX:** Created measures for performance indicators:
   * `Total Sales`
   * `Net Profit`
3. **Interactive Visualization:** Designed a multi-perspective dashboard:
   * **KPI Cards:** For instant high-level metric tracking.
   * **Bar Charts:** To compare sales across product categories and branches.
   * **Line Chart:** To track sales trends over months.
   * **Slicers:** For interactive filtering by branch.
